## EmailJS
Replace these placeholders in script.js:
- YOUR_SERVICE_ID
- YOUR_TEMPLATE_ID
- YOUR_PUBLIC_KEY

## GitHub upload
1. Upload all files to the root of your GitHub repository
2. Replace old files if prompted
3. Commit changes
4. Wait for GitHub Pages to republish
5. Hard refresh your website
